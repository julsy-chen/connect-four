# How To Play:

Connect 4 is a zero-sum game where 2 players take turns dropping disks on a 7 x 6 board until someone wins by connecting 4 of their disks. 

Grab a friend to play with you!

# Blitzcrank

*Blitzcrank* is a bot that you can choose to play against (as Player 1 or 2) which uses alpha-beta pruning to make decisions as to where to drop its disk. Try to beat it!

# Instructions

1. Click on the green <> Code button
2. Copy HTTPS web URL
3. Type 'git clone [url] into Terminal
4. Running using python game.py
- For Mac users: python3 game.py
